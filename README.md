# Programming Paradigms — example labs and exercises

This repository is a collection of small labs and example programs illustrating different programming paradigms: procedural, object-oriented, functional, concurrent, and actor-based. The examples are written in Java and Scala and are intended as educational material for hands-on exercises.

Repository structure
- `list2/` — Scala examples and exercises (LabList2.scala)
- `list3/` — simple demonstration program (main.scala)
- `list4/` — additional list-related lab (Lab4List.scala)
- `list5/` — data structure implementations and examples in Java and Scala
	- `list5/Java/` — `MyStack`, `MyCircularQueue`, `MyLinkedListQueue`, `Node` and example `Main.java`
	- `list5/Scala/` — Scala examples (`main.scala`)
- `list6-coffee-machine/` — simple coffee machine simulator (Java)
- `list7-multithreading/` — concurrency examples and the dining philosophers problem (Java)
- `list8-akka/` — actor-model examples using Akka (Scala)
